# Learning Ratatui

The goal of this repo is to learn how to create a CLI application using Ratatui.
Every step I take to learn this new framework will be documented in the `examples` directory,
from the basics to a simple interactive TUI.

The goal here is to learn by doing — experimenting and messing around with the code.

---

Each file in the `examples` directory can be run and will start a CLI application. To run them, use the following command:

```
cargo run --example ${name of the file}
```

For example:
```
cargo run --example 01
```
